# repo-ventas-2024
# Análisis de ventas 2024 — UTN TUP Organización Empresarial

## Descripción
Análisis estadístico de datos de ventas comerciales correspondientes al año 2024.
Trabajo Práctico — Organización Empresarial — UTN TUP — 2026.

## Escenario
Escenario B — Análisis de Ventas de una Pequeña Empresa.

## Integrantes
- Villanueva, Mauro (P1 — Líder / P2 — Desarrollador / P3 — QA)

## Dataset
Archivo `datos/ventas.csv` — 56 transacciones con producto, cantidad,
precio unitario y fecha de venta (2024).

## Estructura del repositorio

repo-ventas-2024/
├── datos/ventas.csv
├── scripts/analisis_ventas.py
├── resultados/
├── README.md
└── .gitignore

## Cómo ejecutar el script
1. Abrir Google Colab
2. Clonar este repositorio
3. Ejecutar `scripts/analisis_ventas.py`
4. Los resultados se guardan automáticamente en `/resultados/`

## Dependencias
- Python 3.x
- pandas
- matplotlib
